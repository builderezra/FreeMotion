# Overnight polish log
One line per shipped iteration — newest at the bottom. (Started when Ezra went to bed.)

- v2.31 — uid() now appends a random suffix: layer ids from different sessions can no longer collide, which could silently cross-link/destroy media between projects in the shared IndexedDB store.
- v2.32 — audio-only clips (mp3/wav) now show a music-note thumbnail in the timeline instead of a blank box.
- v2.33 — group selection box now hugs the members' actual bounds (was a meaningless 100px square at the corner); tapping grouped content on canvas selects the group as one object, and inside Edit Group the same tap picks the individual member.
- v2.34 — effect search now matches category names and ids too: typing "3d", "warp" or "blur" surfaces the whole family, not just effects with the word in their title.
- v2.35 — Wiggle/Drift/Orbit/Tiles/Raster-Extrude skip the per-frame full-canvas pixel scan they never used: those effects now cost a fraction of what they did during playback.
- v2.36 — home-screen project cards show their layer count in the meta line (fills in as each project is next opened/saved).
- v2.37 — hardened the boot media-cleanup against a race: it now stands down while template/duplicate media writes are in flight and re-verifies every candidate right before deleting, so fast actions at startup can never lose a clip.
- v2.38 — groups gained Color & Fill (full colour grade) and Border & Shadow (silhouette outline + drop shadow) — property parity with normal layers complete.
- v2.39 — add-menu rebuilt to match Alight Motion: AI Scene moved into Media, shapes now PAGE SIDEWAYS with page dots (no vertical scroll), tabs are Shape/Media/Audio/Object/Template/Freehand Drawing with Vector Drawing + Text on the right rail. NEW: Freehand Drawing (draw a brush stroke on the canvas) and Vector Drawing (tap points → filled shape), both real path layers you can restyle.
- v2.40 — Freehand Drawing moved off the hidden top tab onto the instant rail beside Text and Vector Drawing, so it's findable on mobile and all three spawn-instantly tools share one row on PC (like AM).
- v2.41 — shapes now come out identical in every project format: the default box is square (sized off the shorter side) instead of width/3 × height/3, which used to stretch a circle into an oval in 16:9 / 9:16.
- v2.42 — new clips (shapes, text, drawings, imported media) now spawn AT THE PLAYHEAD instead of at 0, and the timeline length auto-fits the clips: it grows when a clip extends past the end and shrinks when clips end earlier (e.g. 3s of clips = a 3s timeline).
- v2.43 — inspector cleanup: removed the redundant header row (thumbnail/rename/duplicate/delete); the orphan gap in the category grid is gone (a lone card now spans full width on PC); delete moved to the top bar next to ⋯/Export; and the top-bar name field now shows the selected layer's name (rename it there) and reverts to the project name when nothing's selected.
- v2.44 — added an Ungroup entry to the ⋯ three-dots menu (shows only when a group is selected); fixed the glitch where tapping a layer scrolled the timeline back to the top (vertical scroll is now preserved across the rebuild).
- v2.45 — reworked Color & Fill to match Alight Motion exactly: a 4-tab rail (None / Solid / Gradient / Media). Solid now fills the whole shape with one flat colour (swatch palette + hex + fill-opacity) — no more accidental gradient. Gradient is its own tab (Linear/Radial/Angular + live preview bar + two stops). Media fills the shape with a picture (cover-fit, clipped, stored as a downscaled data URL so it survives reload). Fill/gradient moved out of "Edit Shape" (now geometry+stroke only); text gets Solid+Gradient; lines/arcs just get one Colour tab. Old projects render unchanged (fill mode is derived from legacy fields).
- v2.46 — Color & Fill nailed down: (1) picked colours now render EXACTLY (a stale hue/sat grade from the old colour-wheel panel was silently shifting every fill — grades no longer touch fill-owned colours); (2) colour KEYFRAMING — a ◇ diamond next to the hex readout animates fill/text colour over time (channel-lerped, shows/deletes on the timeline like any keyframe, auto-keys once armed); (3) fill works on EVERYTHING — videos/images get the same None/Solid/Gradient/Media tabs and a solid pick fully overwrites the clip (None restores it; Colour Tune/grade tools stay available under None); groups recolour their whole silhouette; (4) shapes spawn with natural proportions (rectangle 3:2, arrow/line/semicircle elongated, circle/star square) and identical sizes in every project format.
- v2.47 — the massive one: (1) 37 new AM shapes (speech, moon, snowflake, shield, check, droplet, cloud, play, spiral, sparkle, stamp, bolt, puzzle, pushpin, flag, thumbs-up, paper plane, house, laurel, bookmark, pointing hand, flame, banner, wreath, diamond, plane, umbrella, bomb, boat, magnifier, key, sun, person, rocket, envelope, woman, car) — all in the add menu with icons generated from their real geometry; (2) EVERY shape has editable points — "✎ Edit points" in Edit Shape converts any shape to a path with draggable vertices on the canvas (drag to reshape, tap a ring to add a point, double-tap to delete); (3) Blending & Opacity rebuilt AM-style — opacity slider on top, then Normal/Darken/Lighten/Contrast/Difference/Color/Mask as expandable dropdowns (one mode each for now; Difference has Exclusion, Mask has include/exclude which truly mask the layers below — verified pixel-exact); (4) PC inspector: Effects is now the big full-width card at the bottom, everything else is a square (no more giant Blending card, no orphan gaps).
- v2.48 — shape grid is icon-only (names moved to tooltips): bigger 34px icons, 5-across on phone / 6-across on PC like AM. Upkeep sweep on top: paste-style no longer shares one colour-keyframe object across layers; nulls/adjustments/audio-only clips stop offering Color & Fill / Border cards that couldn't render; spiral now shows the single Colour tab (it's a stroked line, gradients did nothing); the point editor cleanly dismisses itself if its layer is deleted or the project switches mid-edit. Verified: full save→reload round trip of every new field (edited-path subs, angular gradient, fill opacity, colour keyframes, mask blend), duplicate carries them without aliasing, console clean.
- v2.49 — Motion Blur (Content): a NEW effect that blurs what moves INSIDE the clip (real optical-flow motion blur, like RSMB / AE Pixel Motion Blur) — a swinging sword now streaks across the frame while the background stays razor sharp. Four styles: Pixel (per-pixel directional blur along a block-matched flow field, coarse+fine search reaching very fast motion), Smear (one dominant direction, moving areas only), Echo (long-exposure light trails, AE-style lighten merge), Blend (frame blending — rescues low-fps clips). Settings: Style, Shutter, Quality, Threshold, Softness. Featured in the effects carousel; works on video, images, shapes, text and groups; identical in export. Verified: symmetric 0→128→0 smear profile across a moving object's path, static content untouched, decaying echo trail, ~19ms/frame preview cost.
- v2.50 — five fixes: (1) grabbing/scrolling/wheeling the timeline while playing now PAUSES playback (detected on the raw input — the playhead's own auto-scroll was swallowing user scrolls); (2) effect-detail sliders no longer get "stuck to the mouse" after release (the drag-back-to-zero bug — fxScrubber was the one scrubber missing the buttons-released guard); (3) Motion Blur (Content) un-broken: repaints at a paused playhead (every slider tweak causes one) collapsed the blur to a sharp frame, so the effect looked dead while you adjusted it — a two-slot plate cache keeps the blur visible; (4) home screen orders projects most-recently-edited first; (5) pointing-hand shape redrawn (thumb, extended finger, three knuckles — actually looks like ☞); (6) timeline paint-select is now a RANGE — drag down to add rows, drag back the way you came and they unselect again, no manual deselecting.
